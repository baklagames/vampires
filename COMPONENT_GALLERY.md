# Component Gallery

This document defines the DS component gallery for the MVP prototype.

## Usage
- Gallery state builder: `src/ui/screens/component-gallery.ts`
- Entry screen id: `component-gallery` (see `src/ui/navigation-controller.ts`)

## Included Components
- Buttons: ButtonPrimary, ButtonSecondary, ButtonIcon
- HUDChip
- TextField
- ToggleSwitch
- Slider
- ProgressBar
- HeatIndicator
- Toast (queue)
- PixelCard
- ListItem
- ModalSheet
- BottomActionBar
